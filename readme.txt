Thanks for downloading Status-PY

Here are some informations for runing this python code


First of all you need to install the requirements with the "requirements_installer.py" with command:    "sudo python3 requirements_installer.py"


  First you need to install requirements for the code, here is the command for it:     "sudo pip3 install -r requirements.txt"

    Next you need to run the python code by using sudo bcs there are some fetures that need to be in root to work:      "sudo python3 status.py"
    
      Next step is only if you want to make the python code executable then follow these steps:     "sudo apt install pyinstaller" then "pyinstaller --onefile status.py"
        Then just run the executable with:    "./status"
